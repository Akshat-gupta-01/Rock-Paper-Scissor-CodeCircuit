# Rock Paper Scissors (Web-Based)

A simple Rock Paper Scissors game built using Flask (Python), HTML, CSS, and JavaScript.

## How to Run
1. Install Flask:
   ```bash
   pip install flask
   ```

2. Run the app:
   ```bash
   python app.py
   ```

3. Open your browser and go to:
   ```
   http://127.0.0.1:5000
   ```

Enjoy the game!
